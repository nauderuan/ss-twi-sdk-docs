twilioConversation

twilioConversation: null | Conversation

The twilio conversation object instance. This is used to track things such as last read message index. We can do that since this object is passed by reference and thus updated in real time by twilio. For inbox this will be the neo conversation object and for chatterbox this will be the user conversation object. For inbox this is only used to observe the conversation messages and events but not to interact with it. For chatterbox this is used to interact with the conversation.

  • Defined in src/chatConversationMessages/types/types.ts:255
